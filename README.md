# 设计原则
* 劣势互补：用大量的浮点计算弥补C语言高效的缺陷  
* 勇于挑战：复杂的矩阵运算？一个for不行就两个  
* 得过且过：注释什么的随便搞搞就好了，反正CPU看不懂  
* 声东击西：你以为我是来学神经网络的，其实我是来练习C语法的  
* 爱我中华：中文是最好的语言
